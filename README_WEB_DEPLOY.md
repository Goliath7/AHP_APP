# Веб-версия и бесплатный деплой

## Локальный запуск

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```

## Полностью бесплатный деплой (Streamlit Community Cloud)

1. Загрузите проект в GitHub-репозиторий.
2. Перейдите на [share.streamlit.io](https://share.streamlit.io/).
3. Нажмите `New app`.
4. Выберите репозиторий и файл запуска: `streamlit_app.py`.
5. Нажмите `Deploy`.

После деплоя приложение будет доступно по публичной ссылке бесплатно.
